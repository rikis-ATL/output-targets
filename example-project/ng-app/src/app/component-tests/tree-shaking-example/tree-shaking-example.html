Bundle Size Optimization:
This example imports only MyButton and MyCounter components:
import {{ '{' }} MyButton {{ '}' }} from 'component-library-angular/components/my-button';
import {{ '{' }} MyCounter {{ '}' }} from 'component-library-angular/components/my-counter';

This results in a much smaller bundle compared to importing the entire library:
